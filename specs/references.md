## Notas importantes

- Los route modifiers de Google sesgan la ruta, pero pueden no eliminar totalmente peajes, autovías o ferris si no hay alternativa razonable.
- Weather API permite previsión horaria hasta 240 horas.
- Revisar condiciones de caché/almacenamiento antes de persistir datos externos.
